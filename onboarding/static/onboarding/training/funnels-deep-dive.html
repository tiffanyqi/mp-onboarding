Funnels Deep Dive

Guiding Questions

  • Which events do we “count”?
  • What is the difference between filtering events with specific properties when building the funnel, and segmenting/specifying for a property in the ‘Overview Table’ after the Funnel is built?
  • Be able to define the following concepts:
    • Loose ordering
    • Conversion window
    • Changing properties
    • Repeat events
  • Adjustable Funnel conversion windows? (Whenever there is something really custom you want to do, or something you want to build within a custom report, chances are the API or one of its many endpoints can get that for you.)
  • How do properties work? Which ones do we include - First touch, last touch?
  • What is backfilling?
  • 2-second grace period?
  • Loose ordering?
