  • Why do people use custom reports
    • Custom visualizations of existing Mixpanel data
    • Combining Mixpanel data from multiple endpoints
    • Dashboards for business specific KPIs
  • How do you create a custom report
    • “Applications” platform in Mixpanel, choose “Create New”
    • Once created, you will see the live report code feeding in from Github as well as some menu options to perform operations on the report
    • Set to private by default, you can make public so other project members can view
    • Editing within Mixpanel is limited to a single account, but there is the option to develop locally for team coding
    • Debugging is the same steps as with a user’s external site
    • Look for errors in JS console and check the data going in/out through Network XHR tab
